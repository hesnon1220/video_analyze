自動剪歌姬 版本0.0.1  
  
自動剪歌步驟  
    1.對影像進行灰階直方圖運算、紀錄變化量  
    2.依照變化量進行畫面分割  
    3.使用影像辨識模型進行畫面挑選  
    4.紀錄畫面色階  
    5.需手動進行歌詞時間軸  
    6.依照時間軸區間進行畫面填充  
    7.使用第三方軟體新增音樂字幕  
第三方軟體  
    1.demucs 音源分割  
    2.yolov5 影像辨識  
    3.labelGo 半自動標記成式(需手動修正程式碼)  
    4.RhythmicaLyrics 手動標註音節  
    5.ニコカラメーカー2 卡拉OK字幕
